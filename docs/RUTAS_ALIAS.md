# URL Aliases - Spanish Routes (Rutas en Español)

This document tracks all Spanish URL aliases for SEO purposes. Both English and Spanish URLs work and display the same content.

## How It Works

- **English URL** (canonical): `/platform/tool-name`
- **Spanish URL** (alias): `/platform/nombre-español`

Both URLs display the **same page** (no redirects). This is implemented via `next.config.ts` rewrites.

## Spanish URL Naming Conventions

### Generators
- `generator` → `generador`
- Examples: `bio-generator` → `generador-bio`

### Calculators
- `calculator` → `calculadora`
- Examples: `coins-calculator` → `calculadora-monedas`

### Writers/Makers
- `writer` → `escritor`
- `maker` → `creador`
- Examples: `script-writer` → `escritor-de-guiones`

### Ideas
- `ideas` → `ideas`
- Examples: `video-ideas` → `ideas-de-videos`

### Common Terms
- `hook` → `gancho` / `ganchos`
- `hashtag` → `hashtag` (same)
- `username` → `nombre` / `nombres`
- `caption` → `subtítulo` / `subtitulos`
- `description` → `descripción` / `descripciones`
- `title` → `título` / `titulos`
